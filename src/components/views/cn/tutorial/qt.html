<%- LANG('导入私钥') %>

<%- LANG('UB的所有代币均记录在对应的地址上，而一个私钥对应一个地址，因此私钥是操作UB最重要的信息。') %>

<%- LANG('比特币QT钱包导出私钥') %>

<%- LANG("比特币的qt钱包导出的钱包文件是一个以'.dat'结尾的文件。UB钱包兼容比特币的导出文件，因此恢复方式和比特币一样。只要先停止UB钱包，然后把备份文件更名为'wallet.dat'，并复制到数据目录下的'wallets'目录下，覆盖原有的文件。然后重新启动UB钱包，就可以看到导入的地址。") %>

<%- LANG('其他轻钱包导出私钥') %>

<%- LANG('轻钱包的实现方式比较多样，因此这里无法列出所有。本文以electrum为例，其他轻钱包可以借鉴，或者咨询钱包开发者。') %>

<%- LANG("首先，在electrum中选择'view'菜单下的'show addresses'项调出'addresses'窗口") %>

<%- LANG("然后在地址上右键单击，选择'private key'") %>

<%- LANG("此时可以从弹出窗口看到该地址对应的私钥") %>

<%- LANG("把该私钥导入到UB钱包，下面的操作均是在UB钱包中进行。") %>

<%- LANG("首先通过菜单项'帮助'->'调试窗口'打开UB钱包的控制台。并输入命令导入私钥：") %>

<%- LANG("命令格式为：") %>
<%- LANG("Imortprivkey [私钥] [账户] [rescan]") %>
<%- LANG("私钥必须输入，账户可以不输人，rescan标志输入'false'可以避免导入时扫描钱包。注意如果输入rescan标志，则账户也必须输入。如不知道用什么账户可以输入''表示默认账户。") %>

<%- LANG("导入私钥以后用validateaddress命令检查是否导入成功。该命令的参数为私钥对应的地址，可以从electrum钱包获取。") %>

<%- LANG("此时私钥就导入成功了，等钱包数据完全同步以后，就能看到账户中有余额。") %>

<%- LANG("注意：") %>
