資產啟動

UnitedBitcoin沒有預挖礦，比特幣區塊鏈上的所有活躍的比特幣將在UnitedBitcoin區塊鏈上獲得同等數額的UnitedBitcoin。活躍地址和非活躍地址的定義見"領取UnitedBitcoin"頁面上的定義。

生態系統支持與穩定性

UnitedBitcoin基金會管理比特幣網絡中不活躍地址的比特幣，其中一部分資金將被鎖定，並錨定法定貨幣，部分資金將用於支持加密貨幣的生態系統。

快速交易

區塊容量新增到8MB和支持隔離見證可以大大提高鏈上的可伸縮性。UnitedBitcoin也將在2018年新增對閃電網絡的支持。

重放保護

新的交易簽名提供重放保護，從而使比特幣區塊鏈上的交易不能在UnitedBitcoin區塊鏈上重放，反之亦然。

更改地址格式

為了防止用戶意外地將UnitedBitcoin提現到比特幣地址，UnitedBitcoin已經改變交易的地址格式。

智慧合約

智慧合約現時正處於測試階段，將被加入到UnitedBitcoin的交易中，並將為整個網絡帶來全新的可能性。

閃電網絡

為了支持支付通路和進一步提高鏈上的可伸縮性，UnitedBitcoin將在2018年上半年實施閃電網絡。
